It is a simple rock, paper and scissors game which should be very familar for all people.

Description:
1. The game is defaulted to play with CPU.
2. User input rock, paper or scissors. Then CPU will randomly choose one of the three output by random library.
3. Program will judge whether it is a win/lose/tie between user and CPU.
4. Play again button is implemented. User can play again without reloading the document.

3. Using random library to generate random number.
