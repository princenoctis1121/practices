It is a number guessing game which should be very familar for all people.

Description:
1. The game start by asking user to type the maximum number you want for the game.
2. A clue is randomly generated by random library. Then user can start guessing the number.
3. The terminal line will guide you whether the number user guessed is larger or smaller than the clue.
4. Play again button is implemented. User can play again without reloading the document.

3. Using random library to generate random number.
