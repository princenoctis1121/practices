This is a password manager program help user to save passwords.
*It is not save in terms of security for normal use so it is just for coding practice.

Description:
1. The program is to help user store and view passwords.
2. All adata is protected by a master password that is required to create when the program is initialized. 
3. User can choose from add mode or view mode to add or see password.
4. User can add password in add mode by entering their name and password.
5. User can view the password in view mode by entering the master password and name.
